# 后端作为 Tauri 外壳的子进程运行

桌面端应用（`desktop-tauri/`）启动时，会拉起 FastAPI 后端作为本地 Python 子进程。前端通过 HTTP 与后端通信，默认端口为 `8000`。

Tauri 命令只负责桌面外壳相关能力，例如文件选择器、系统托盘和更新入口。对话、工具、代理和模型服务商逻辑都放在后端，而不是放在 Rust 代码中。

这样设计的原因：

- 同一套后端也服务远程访问场景，桌面端和移动网页端共用一条逻辑路径。
- Python 更适合承载大模型、模型服务商和工具生态。
- 子进程边界让后端可以打包成独立二进制，也方便在没有桌面外壳的情况下做测试。
