# suyo v1.0 开发与发布计划

状态：`v1.0.0-rc.1` 候选提交准备中；macOS 主机源码回归与 Apple Silicon 本地包预检通过，五个原生目标、受控 Beta 与公开验收待完成
目标版本：`1.0.0`
发布主题：安全可交付的本地办公 Agent
范围冻结日期：2026-07-14

## 完成定义

v1.0 不是 WorkBuddy 全量能力对齐版。v1.0 完成必须同时满足以下条件：

1. Windows x64、macOS arm64/x64、Linux x64/ARM64 的正式安装包从同一 tag 构建，完成安装、
   启动、正常退出和无孤儿进程验证；稳定 macOS 包必须使用 Developer ID 签名并通过 Apple 公证。
2. DOCX、XLSX、PPTX 在所有桌面平台通过受限声明式工具完成创建和有限编辑；不得为此开放任意
   Python、Shell 或 Office GUI 自动化。
3. AI 通过声明式工具修改已有文件前产生持久、校验和可验证的本地版本；保存使用临时文件与原子
   替换，失败时原文件保持不变；用户可以列出和恢复版本。Linux Bash/Python 使用 Bubblewrap、
   macOS 使用 Seatbelt，且必须先在私有事务副本执行。Windows Bash/Python 使用 Job Object 回收
   进程树但直接作用于已批准工作区，不具备文件系统或网络隔离，也不承诺失败回滚；产品和发布说明
   必须持续明确这一差异。
4. 一个图片生成 Provider 完成凭据、费用提示、异步状态、取消、超时、失败恢复、产物保存和预览闭环。
5. 腾讯文档官方 MCP 完成受控原始 Token 鉴权、工具过滤、写操作权限、凭据存储和真实账号 E2E。
6. 来源级能力策略、持久审计和急停覆盖桌面、调度器、连接器以及任何进入 Beta 的外部来源。
7. 完整后端、前端、Rust、Playwright、供应链审计及原生安装包门禁全部通过；不存在未关闭的
   P0/P1、安全边界绕过或已知数据损坏。

## Office v1 支持边界

### DOCX

- 创建标题、段落、项目符号、表格、分页和图片；
- 对既有文档追加内容、替换明确文本、增加表格或分页；
- 不支持宏、修订/批注保真、外部模板、嵌入对象和像素级排版承诺。

### XLSX

- 创建工作表、写入二维数据、公式和基础数字/字体/填充样式；
- 对明确工作表和单元格区域更新、追加行、增删工作表；
- 只写公式，不声称在本地完成 Excel 公式重算；不支持宏、外部链接和任意二进制对象。

### PPTX

- 创建标题页、标题与正文页、文本框、表格和本地图片；
- 对明确页面追加或替换文本、增加页面；
- 不支持宏、复杂动画、嵌入对象、母版级编辑或像素级保真。

所有格式共同要求：工作区约束、扩展名白名单、符号链接逃逸防护、输入大小预算、写入前版本、
临时文件、重开验证和原子替换。

## 腾讯文档 v1 支持边界

- 固定官方端点 `https://docs.qq.com/openapi/mcp`；
- 只允许产品定义的 `Authorization` 原始 Token 模式，禁止任意凭据 Header 名称和值模板；
- Token 进入现有 CredentialStore，不写入配置、日志、工具结果或模型上下文；
- 默认开放查询、搜索和读取；创建、追加与更新服从写操作权限；删除节点默认不开放；
- 真实 Token 缺失时运行严格 mock 合同测试，但真实账号 E2E 是 GA 前置证据。

## 图片生成 v1 支持边界

- 只实现一个明确的 Provider/协议；v1 不承诺视频、多供应商路由、局部重绘或参考图编辑；
- 每个请求在网络发送前持久记录 call ID、模型、尺寸、参数摘要、输出路径和计费状态，不记录密钥或提示词；
- 支持取消、超时、大小限制、安全文件名和工作区内保存；
- 同一 call ID 只允许提交一次；超时、取消或下载失败标记为 `uncertain`，不得自动重试，用户核对账单并
  再次确认后才能创建新的计费请求；
- 失败不得留下伪成功产物，PNG 必须完成容器与像素流重开校验；
- 生成前展示或返回可见的供应商/模型与费用风险提示。

## Stretch 与明确不进入 v1.0 的范围

只有核心范围完成且没有消耗 RC 缓冲时，才允许加入规则型 Auto Router 或 HMAC Webhook Beta。
单一 IM 远程通道若进入 Beta，必须替换其他 Stretch，不能叠加承诺。

以下能力不属于 v1.0：Teams/RBAC、共享项目云盘、多租户云 Agent Runtime、无 PC 云执行、
原生移动客户端、公开云分享、Ardot 等价专业画布、正式视频生成、完整中国办公连接器套件。

## 停止和降级条件

- 任一命令路径绕过声明的平台边界（Linux Bubblewrap、macOS Seatbelt、Windows Job Object 与已批准
  工作区直接执行披露），或把 Windows 直接执行描述成隔离/可回滚：停止该实现并重做架构；
- Office 写入出现静默损坏、路径逃逸或不可恢复覆盖：停止功能扩展，先修复数据安全；
- 腾讯文档真实 Token 或图片 Provider 测试凭据在 RC 前不可用：该能力不得标记 GA；
- 稳定 macOS tag 缺少 Developer ID 或 Apple 公证凭据：发布工作流必须失败；
- 功能冻结后仍有 P0/P1、供应链 vulnerability、权限绕过或迁移失败：延期 GA，不降低门禁。
